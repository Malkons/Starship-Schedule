# Starship-Scheduler

#### Overview

An app that uses Firebase to populate a schedule with user created inputs.  Uses Moment.js to keep track in real-time the frequency in which the ships arrive. 

#### Directory Layout
```
Starship-Shedule
    - assests
        - css
            - style.css
        - images
        - javascript
            - app.js
    - index.html 
```
